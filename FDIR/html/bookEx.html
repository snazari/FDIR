Unknown Input Observer Dynamic System Matricies

This example is form the book: Robust Model Based Fault Diagnosis for Dynamic Systems by J. Chen Please see the associated ShareLaTeX pdf file "UIO and Examples"

A = [-1 1 0;-1 0 0;0 -1 -1];
B = [-1;0;0];
C = [1 0 0;0 0 1];
D = 0;
E = B;

Construct the state space system

Let us construct the system as a state space object for simulation.

sys = ss(A,B,C,D);

Step one: Check rank

We check to see if the rank(CE) = rank(E) = 1

rank(C*E)
rank(E)

ans =

     1


ans =

     1

Step two: Compute Observer Matricies

Since we have no issues in the previous step, let us compute the observer matrices now:

H = E*inv((C*E)'*(C*E))*(C*E)'
T = eye(3)-H*C
A1 = T*A

H =

     1     0
     0     0
     0     0


T =

     0     0     0
     0     1     0
     0     0     1


A1 =

     0     0     0
    -1     0     0
     0    -1    -1

Step three: check observability

We must check the system A1,C observability

rank(obsv(A1,C))

ans =

     3

Pole Placement

Pole placement is used to assign the observer poles

K1 = place(A',C',[-2,-10,-5])'

K1 =

    6.7231   -3.7816
    7.5581  -11.3458
   -3.0543    8.2769

Finish observer design

Finally, the F and K matricies are computed

F = A1-K1*C
K = K1 + F*H

F =

   -6.7231         0    3.7816
   -8.5581         0   11.3458
    3.0543   -1.0000   -9.2769


K =

         0   -3.7816
   -1.0000  -11.3458
         0    8.2769

x1_0 = 100;
